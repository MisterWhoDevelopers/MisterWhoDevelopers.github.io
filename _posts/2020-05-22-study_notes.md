# 读书笔记

## 书名：《如何有效阅读一本书》时间：2020-05-24 14:19

读书应该带着明确的目的，充满主动性的阅读，消化书中的信息，使之成为自己的东西

深入理解书中的要点或思想，并随时拿来参考

学会利用笔记能更懂得如何选书、购书、读书和活用

为了一次美好的阅读经历，仅仅“读过”是远远不够的，要学会与书相处

1. 如何选书：怎样选择自己真正想读的书？
2. 如何购书：怎样买到对自己真正有益的书？
3. 如何读书：怎样加深理解深入思考
4. 如何活用：怎样运用从书中获得的知识

笔记的使用方法：
1. 随想笔记：记下平时的感悟，这与日后的选书息息相关
2. 购书清单：找到真正想读的书，主动的选择书籍【这么说来，我就不用靠着推荐算法发来的各种推送消息读书了】
3. 各种报道的剪报：尽情收录感兴趣的内容，从中发现“这本书我一定要读”，那么直接写到购书清单上
4. 读书笔记：将读后感转变成文字保存下来，留下你与这本书交流过的证据

由于我们无法预测自己想读的书，所以不要分类，直接把信息都罗列在笔记本上，反而更好

只要坚持就一定有效：好记性不如烂笔头，做笔记不是简简单单的“流水账式操作”，而是将来总有一天能用到的知识储备

把灵感记录在纸上，让思考书面化，原因是可能会忘记灵感

通过笔记养成良好的读书习惯

1. 养成习惯
2. 放大读书笔记的作用
3. 创造别具一格的记录方式【怎样创造别具一格的记录方式呢？《涂鸦笔记》】

更好的读书体验：可以成就全新的自己

1. 选书：记录想读和需要读的书
2. 购书：判断是不是值得买
3. 读书：做好写读书笔记的准备，重要部分提炼出来
4. 记录：制作读书笔记，留下与书交流证据
5. 活用：再次浏览读书笔记，在检索和参考的过程中，帮助自己获得智慧得到提升，也方便自己日后重读

一元化的有点：

- 简单易懂，人人都能看懂做到
- 更容易坚持，自由度高，可自行设计
- 信息存在同一地方
- 直接记录、参考，不易混淆
- 因为“杂乱无章”，所以更方便“活用”

1. 可以书写任何内容
2. 按时间顺序书写
3. 附上日期
4. 灵活运用速记或略记：简写，利用更少的短语，略记：“首字母缩写”等
5. 可以粘粘

## 书名：《如何有效阅读一本书》作者：奥野宣之 时间：2020-05-25 20:36

确认好清单，如果觉得“这本书不买也行”，就把书名整个划掉，如果买下了，就在旁边标上“已买”

潜水艇有“主动声纳”和“被动声纳”两种声纳系统，主动声纳是靠自身发出声波的反射状态来进行计量的，而被动声纳系统则是靠接收其他船只或潜水艇发出的声波来活动的。也许你觉得主动声纳听起来更厉害，但得到广泛使用的恰恰是被动声纳系统。

我们常说“要对所有事物抱有好奇心”。【好奇心不是害死猫吗？查完资料发现这是探索发现的求知欲，不然活着很没意思，作者表达的意思是对书中内容探索发现】好奇心固然很重要，但不随时记录下来，也只会是一个漫不经心的小想法。瞬间的思考就像沸水里的气泡，只会出现一瞬间。如果马上把想法记下来，就算字迹再潦草，也比什么都不记好得多。以后看到这段笔记，可以从这个小小的想法展开思考，说不定能得到一个绝妙的创意

一切都应该以“能坚持下去”为先

人们经常说“你为别人讲解书中的内容时，才会真正理解它”

当你以思想输出为前提去读书时，思想输入的质量也会有所提升，而且亲手写文章的好处比口头叙述要多

写读书笔记也是有窍门的，简单来讲，就是要彻底的专注于“对自己很重要的事情”

读书的目的是用自己的方式学习，而不是模仿评论家的做法，学习、吸收对自己真正有用的智慧和语句才是最重要的

在记录感想的时候，不要写一些揣测的或者外来的想法，而是要写下自己的心声，也就是感受和想法，这样会让读书笔记更有内涵

筛选程序
1. 通读：一边阅读，一边把觉得有价值的那一页折角
2. 重读：读完一遍之后，再把折角的几页重读一遍。如果仍然觉得好，就把那一页另一角也折起来
3. 标记：再重新读一遍折起来上下两角的几页，如果第三次阅读后仍然值得一读，就用笔在上面做记号。接下来，再读一遍第3步中的内容，把最后还是无法舍弃的内容抄在读书笔记上就可以了

可以用以下不同的画线方式对标记的对象进行区分：
- 普通直线：用于“重要”“客观上很重要”“暂时先画出”的情况
- 波浪线：用于“非常重要”"主观觉得值得重视"“想记住这里，以后加以运用”的情况
- 圆圈：用来圈出重要的专有名词、关键字、关键句等需要吸引眼球的内容

写读书笔记之前，要注意以下三项信息：
- 写读书笔记的日期
- 书名
- 作者名

还有两个要素
- 对自己来说重要的内容（摘抄）
- 自己对这篇文章的感想（评论）

## 书名：《如何有效阅读一本书》作者：奥野宣之 时间：2020-06-08 18:43

摘抄对自己来说很重要的文章和写主观评论
摘抄的第一个好处：促进、加深对书的记忆
摘抄的第二个好处：加深理解
一定要摘抄让自己心动的语句，觉得这么一说确实有道理的语句

亚瑟·叔本华在《论读书》中写道：“如果你觉得读书就是为了模仿别人的想法，那么这是思想上的懒惰。请丢开书本自己思考”

读书笔记是自己和书的对话，所以不需要别人的观点，但如果在做完读书笔记以后重读书评，就可以参考“第三方的观点”，换个角度审视自己的这次读书体验，于是你会发现，“原来对自己来说有着某种意义的书，在这个人眼里是这样的”

“所谓创意，只是把原有的元素重新组合而已”，美国广告大师詹姆斯·韦伯·扬在《创意的生成》一书中如是说

## 书名：《涂鸦笔记》作者：麦克·罗迪 时间：2020-06-21 19:50

什么是涂鸦笔记？
作者认为多而杂的笔记，很难记忆，晦涩不易重读笔记，而涂鸦笔记只需要专注于倾听，抓重点记录

涂鸦笔记是示意图，它根据以下活动中收集储存的想法、观点而建立：演讲、圆桌对话、经历
这种视觉性和整体的笔记方法用到了你的大脑，因为你可以理解它们，信息输入：观察、思考、聆听，信息输出：涂鸦笔记 大脑和全身共同协作

你创作的涂鸦笔记中融入了你的独特个性
从绘画和书写风格到其中包含的想法，加上将这些想法表达与纸面的放法，结果是什么？每份涂鸦笔记都是独一无二的，尽管其背后的指导原则是相同的

做涂鸦笔记，你需要积极聆听有意义的观点，思考它们是什么意思，然后建立一个直观的示意图，目标不是钻到细节里，而是将聆听到的主要观点转化为直观的笔记，其中包含文字和图片

小孩们用画画来表达他们的想法，他们不会关心他们的画是否完美，只要能表达他们的想法就行了

1. 常规的笔记比较枯燥，所以用手绘笔记来表达重要的观点
2. 解放局限性自由记录内容
3. 涂鸦笔记作为一张图能将听到的和看到的丰富的展现出来
4. 保持一个积极的大脑和身体状态能让你随后还能回忆起细节
5. 涂鸦笔记可以融入个性，使内容更丰富
6. 基本能用方形、圆形、三角形、线条和点画出所有的物体
7. 表达想法，而非艺术【画的再烂也是自己想表达某个想法的图像】
8. 一步一步来才能成功

1. 双重编码理论认为，我们的大脑使用语言和视觉两个频道来处理信息
2. 使用双频道能让我们创造更加跨界的视觉图
3. 在几天、几个月甚至几年后，视觉图仍可以帮你集中注意力，因为当你全身心工作时就没有机会分心了
4. 不断地练习做涂鸦笔记，慢慢地就会感到越来越自然
5. 做涂鸦笔记能让你全神贯注
6. 比起详细的文字描述，绘制简单的图画更快更有效
7. 做涂鸦笔记能使你变得轻松；不拘泥于细节，而是把握大方向

1. 做涂鸦笔记始于听清信息
2. 当你保持积极聆听的状态时，你的大脑能找出规律和储藏信息
3. 全神贯注
4. 排除干扰
5. 沉静于演讲中
6. 储藏信息
7. 找出规律
8. 利用一切可能的机会锻炼倾听技能
